# Doctors Database (Doctor Who) – WinForms + Entity Framework

A Windows Forms (.NET Framework) application that displays **Doctor Who** data from a SQL Server database using **Entity Framework (Database-First)** and **LINQ**.

---

## What the app does

- Select a Doctor (1–13) from a ComboBox
- Shows Doctor details (Actor, Series, Age, First Episode title, Year)
- Loads the Doctor photo stored in the database (`byte[]`) and displays it in a PictureBox
- Shows all companions for the selected Doctor in a multi-line ListBox
- Includes a menu: **File → Exit** (shortcut **Ctrl + Q**)
